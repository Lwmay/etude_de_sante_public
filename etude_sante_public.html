Etude de santé publique¶

Food and Agriculture Organization of the United Nations http://www.fao.org/home/fr/

Objectifs de l'étude¶

L'tude porte sur la mise en vidence des principaux indicateurs pour l'anne 2017 :

  • Proportion de personnes en tat de sous-nutrition
  • Pays pour lesquels la proportion de personnes sous-alimentes est la plus forte.
  • Nombre thorique de personnes qui pourraient tre nourries.
  • Nombre thorique de personnes qui pourraient tre nourries avec uniquement des produits d'origine vgtaux.
  • Utilisation de la disponibilit intrieure.
  • Pays ayant le plus/le moins de disponibilit/habitant.
  • Pays qui ont le plus bnfici daide depuis 2013.

# import csv file
def import_csv_to_dataframe(filename):
    try:
        return pd.read_csv(DATA_PATH + filename, encoding='utf8')
    except Exception as e:
        print(">>> Le fichier {} est introuvable.".format(filename))
        sys.exit(e)
        
# calculte pourcentage
def prop(nb_1, nb_2):
    return round(nb_1 / nb_2 * 100, 2)

Remarque sur les données¶

Nous n'avons pas de donnes de disponibilit alimentaire pour tous les pays. Nous allons donc travailler sur un chantillion et appliquer les rsultats sur la population mondiale relle.

Besoins journaliers recommandés¶

Le poids moyen d'un adulte est de 62 kg. (Source - BMC Public Health www.https://bmcpublichealth.biomedcentral.com. )

Le besoin moyen en kcal est de 2500 kcal / jour . (Source - BMC Public Health www.ncbi.nlm.nih.gov/. )

La rfrence nutritionnelle en protines des adultes en bonne sant (RNP) est de 0,83 g / kg / jour . (Source - anses www.anses.fr. )
